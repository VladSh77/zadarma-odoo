Zadarma Integration for CampScout

Повна інтеграція хмарної АТС Zadarma з Odoo CRM

Функціонал

  • 📞 Автоматичне збереження дзвінків — всі вхідні та вихідні дзвінки зберігаються в Odoo через webhook
  • 🔗 Прив'язка до CRM — дзвінки автоматично прив'язуються до контактів і лідів
  • 💬 Chatter нотатки — при кожному дзвінку з'являється запис в історії контакту або ліда
  • 📲 Click-to-Call — кнопка виклику прямо з картки контакту через Zadarma API
  • 📥 Імпорт історії — ретроспективний імпорт дзвінків за будь-який період
  • 🎙️ Запис розмов — посилання на запис відображається в картці дзвінка
  • 👤 Хто дзвонив — SIP номер менеджера автоматично прив'язується до запису
  • CallerID routing — правила вибору номера відправника залежно від країни
  • Пошук по телефону — знаходить контакти незалежно від форматування номера
  • Авто-лід — при дзвінку з невідомого номера автоматично створюється лід
  • Статистика на картці — лічильник дзвінків на картці партнера і ліда

Налаштування

1. Вкажіть API Key та API Secret у Налаштуваннях компанії
2. Пропишіть SIP ID кожному менеджеру у профілі користувача
3. Додайте Webhook URL у кабінеті Zadarma: https://your-domain/zadarma/webhook
