Odoo 17 Module

Fayna Zadarma Telephony

Повна інтеграція хмарної АТС Zadarma з Odoo 17 CRM: автозбереження дзвінків, запис розмов у chatter, click-to-call, SMS через Zadarma API, auto-lead з пропущених дзвінків.

Рішення: Fayna Digital для CampScout Автор: Volodymyr Shevchenko Ліцензія: LGPL-3 Версія: 17.0.1.6.0
Можливості

Що вміє модуль

📞

Auto-log дзвінків

Webhook Zadarma → створення запису у zadarma.call, лінк до res.partner і crm.lead.

🎧

Запис розмов

Автоматичний download MP3 у chatter партнера. Зберігається permanently у Odoo filestore.

☎️

Click-to-call

Кнопка «Подзвонити» у картці партнера — через Zadarma API піднімає дзвінок на внутрішньому extension.

🎯

Auto-lead з пропущених

Пропущений дзвінок з невідомого номера → автоматичний crm.lead з UTM=phone.

📨

SMS через Zadarma

Надсилання SMS з Odoo через Zadarma Send SMS API + статистика доставки per-partner.

👤

Auto-owner detection

Маппінг internal extension → res.users: автоматично визначає, хто з менеджерів вів розмову.

📥

Bulk import

Import історичних дзвінків через Zadarma Statistics API — chunk-based, з progress і resume.

📊

SMS Analytics

Окреме view з розбивкою по успіху / failed / pending, рейтами доставки, історією per-phone.

Потік

Як обробляється вхідний дзвінок

1
Zadarma отримує дзвінок

На номер CampScout → Zadarma PBX → webhook POST на /zadarma/webhook (NOTIFY_START, NOTIFY_END, NOTIFY_RECORD).

2
Partner / lead match

Модуль шукає res.partner за phone (normalized). Якщо не знайшов — створює lead з UTM=phone і assigned-менеджером за extension.

3
Call record + chatter

Створюється zadarma.call; у chatter партнера / lead пишеться повідомлення з тривалістю, напрямком і менеджером.

4
Recording attach (async)

Коли Zadarma готує MP3-запис — окремий webhook повідомляє URL; модуль фетчить і attach як ir.attachment до call-запису + повідомлення в chatter.

Встановлення

Конфігурація

1
Odoo-модулі

base, crm, mail, sms.

2
Zadarma credentials

User Key + Secret з my.zadarma.com. Налаштування: Settings → Companies → Zadarma.

3
Webhook URL

У Zadarma CRM → Integrations: вказати https://your-odoo.com/zadarma/webhook. Підписати webhook secret у Settings Odoo.
